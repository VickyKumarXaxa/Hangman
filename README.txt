To run game, visit the game file directory in terminal and enter the following command

perl Hangman.pl

Game Instructions :-
The Program will select a random word and display underscores equivalent to the length of the word.
The Game Life/Chances left to the player will also be displayed both in text and as "Hangman".
The Player enters the letter when asked to make a guess.
If Player enters more than one letters, the program will only accept the first one.

If the entered letter is present in the word, then it is displayed in its correct position(s).

The game ends when the word is complete or all lives end.
At the end the actual word is displayed along with the result.

After the game finishes the program asks the user to enter Y/N to continue to next game or end the match.
